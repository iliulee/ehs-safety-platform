### 任务 1：工人「复工」按钮（问题 1）
目标：误操作"离场"后能恢复在岗。

代码改动点（仅这 3 个文件，不要动其他文件）：
1. `src/pages/workers/components/WorkerCard.tsx`
   - 在 WorkerCardProps 接口增加 `onReactivate?: () => void`（可选）
   - 在函数形参解构增加 `onReactivate`
   - 把原本"离场"按钮的逻辑（status === 'left' 显示的红色"删除"按钮）
     改为：当 `status === 'left' || status === 'suspended'` 时显示绿色"复工"按钮，
     onClick 调用 onReactivate

2. `src/pages/workers/components/WorkerList.tsx`
   - 在 WorkerListProps 接口增加 `onReactivate: (worker: Worker) => void`
   - 在函数形参解构增加 `onReactivate`
   - 在 WorkerCard 调用处增加 `onReactivate={() => onReactivate(w)}`

3. `src/pages/workers/WorkerListPage.tsx`
   - 在组件内增加 `handleReactivate` 函数：
     ```typescript
     const handleReactivate = async (worker: Worker) => {
       if (!worker.id || !confirm('确定将该工人恢复在岗？')) return
       await workerService.update(worker.id, { status: 'active' })
     }
     ```
   - 在 `<WorkerList ... />` 处增加 `onReactivate={handleReactivate}`
